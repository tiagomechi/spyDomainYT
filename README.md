# Video Scraper

Este projeto é um scraper de vídeos do YouTube baseado em Selenium que captura informações detalhadas sobre os vídeos anunciados em um domínio específico.

## Estrutura do Projeto

project/
│
├── main.py
├── video_scraper/
│ ├── init.py
│ ├── utils.py
│ ├── video_info.py
│ └── file_operations.py
├── requirements.txt
└── README.md
